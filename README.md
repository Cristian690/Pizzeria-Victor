# 🍕 Pizzeria Victor
White-label pizza ordering web application that generates pre-formatted WhatsApp messages for streamlined order processing.
## 📘 About
A fully functional pizza ordering system designed to help pizzerias manage orders efficiently. Customers browse the menu, customize their orders, and the app generates a complete WhatsApp message ready to send to the business.
Currently implemented as a hardcoded single-instance solution, with plans to evolve into a multi-tenant SaaS platform.

## ✨ Features

- 🍕 Dynamic menu display - Browse available pizzas and products
- 🛒 Order customization - Select items, quantities, and preferences
- 💬 WhatsApp integration - Auto-generates formatted order messages
- 📱 Responsive design - Works seamlessly on mobile and desktop
- ⚡ Fast & lightweight - Pure JavaScript, no heavy frameworks

## 🛠️ Tech Stack
Frontend:

- HTML5
- CSS3
- JavaScript (Vanilla)

Integration:

WhatsApp Web API (URL scheme)

## 🚀 Roadmap
This project is designed for scalability:

✅ Phase 1 (Current): Hardcoded single-client implementation
🔄 Phase 2 (Planned): Admin panel for menu management
🎯 Phase 3 (Future): Multi-tenant SaaS platform with white-label customization

## 💡 Use Case
Perfect for small to medium pizzerias that want to:

Reduce phone order errors
Streamline the ordering process
Provide customers with a modern ordering experience
Integrate seamlessly with WhatsApp (most popular in Latin America)
